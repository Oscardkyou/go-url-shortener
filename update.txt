    Что можно улулчшить по этому проекту:

    Хранение данных в базе данных вместо памяти.

    API-интерфейс для взаимодействия.

    Графический интерфейс пользователя.


Утилс:
    gotests: Генерирует тесты для Go.
    gomodifytags: Добавляет или удаляет теги для полей структур в Go, что может быть полезно при работе с библиотеками сериализации.
    impl: Генерирует методы для интерфейса в Go.
    goplay: Позволяет быстро запустить фрагменты кода Go, аналогично playground от Go.
    dlv: Отладчик для языка программирования Go.
    staticcheck: Статический анализатор для кода Go. Проверяет код на наличие общих ошибок и следование рекомендуемым практикам.
    gopls: Является официальным сервером языка для Go. Обеспечивает функциональные возможности IDE, такие как автодополнение, определение и так далее.
    go-outline: Генерирует дерево разбора файла Go в удобном для чтения формате.
